Каталог готелів та пошук турів через IT-Tour API. Вставляйте через шорткоди в Elementor.

== Description ==

Плагін для турагентства Anex Tour. Включає:
* Повний каталог популярних готелів з фільтрами (рейтинг, бюджет, зірки, харчування, пляж, готель, спорт)
* Пошук турів з API IT-Tour
* Форму бронювання з відправкою email та збереженням заявок в адмінці
* Адмін-сторінку зі списком заявок та CSV-експортом

== Встановлення ==

1. Завантажте плагін і розпакуйте в /wp-content/plugins/anex-tour/
2. Активуйте плагін у WordPress Admin → Плагіни
3. Перейдіть у Anex Tour → Налаштування
4. Вставте API Token від IT-Tour
5. Заповніть дані агентства (телефон, адреса, Viber/Telegram)
6. Вставте потрібні окремі шорткоди у блоки Elementor.

== Шорткоди ==

[anex_search]            — форма пошуку з дизайну каталогу; без блоку результатів на сторінці перенаправляє на сторінку каталогу (slug з налаштувань або знайдену автоматично)
[anex_tour_results]      — фільтри та результати пошуку
[anex_tour_filters]      — тільки фільтри пошуку
[anex_tour_results_main] — тільки права частина результатів
[anex_hot_tours]         — блок "Гарячі тури" у DOM-структурі каталогу
[anex_directions]        — популярні напрямки

== Використання в Elementor ==

1. Задайте API Token у Anex Tour → Налаштування
2. Створіть нову сторінку → оберіть шаблон Elementor Canvas (без шапки теми)
3. Перетягніть віджет "Shortcode" → вставте потрібний окремий шорткод, наприклад [anex_search] або [anex_tour_results]
4. Збережіть → опублікуйте
